# MC360 Dashboard v2

Live Demo: https://dashboardd-v2.netlify.app/

## Overview

MC360 Dashboard v2 is an early dashboard concept for Maritime Connect 360. This version establishes the core layout, navigation, and user interface patterns for an enterprise maritime platform.

## Features

- Enterprise dashboard
- Dark interface
- Responsive layout
- Dashboard overview
- Applications screen
- User management
- Material Symbols icons

## Technologies

- HTML5
- CSS3
- JavaScript
